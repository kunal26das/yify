# Yify

A movie browsing app built with Expo and React Native.

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```

3. Run on a platform

   - **Android:** `npm run android`
   - **iOS (Expo Go):** `npm run ios`
   - **iOS (development build):** `npm run ios:dev` — uses native modules like device corner radius
   - **Web:** `npm run web`
